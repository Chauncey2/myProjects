# myProjects
> 项目是对爬取的职位数据进行可视化显示，用到的插件是百度的echarts
***
> 语言：Python

> 框架：Django

> 数据库：MongoDB

> 数据来源：自己编写的职位信息爬虫从国内几个某著名招聘网站中抓取的信息

> 处理逻辑简单，没有用到比较流行的AI基础，就是简单的统计分析，然后可是话（ps：自己坐着玩，小白一个，大神勿喷）

> 可视化控件：百度echarts
